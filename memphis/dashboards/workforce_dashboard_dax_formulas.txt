```dax

```

### Average Efficiency Rate

### Top Performer Name
```dax
Top Performer = 
CALCULATE(
    SELECTEDVALUE(Query1_OperatorPerformance[Operator]),
    TOPN(
        1,
        Query1_OperatorPerformance,
        Query1_OperatorPerformance[ComponentsProcessed],
        DESC
    )
)
```

### Top Performer Components

### Performance Alerts Count
