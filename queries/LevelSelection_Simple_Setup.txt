-- STEP 1: Create Calculated Column in Your Main Table
-- Go to your main table (the one with High_Level, Mid_Level, W_Level columns)
-- Right-click table → New Column
-- Name it: "Selected Level"
-- Paste this DAX (replace 'YourTableName' with your actual table name):

Selected Level = 
VAR SelectedColumn = SELECTEDVALUE(LevelSelection[ColumnName])
RETURN
    SWITCH(
        SelectedColumn,
        "High_Level", 'YourTableName'[High_Level],
        "Mid_Level", 'YourTableName'[Mid_Level],
        "W_Level", 'YourTableName'[W_Level],
        'YourTableName'[High_Level]
    )
